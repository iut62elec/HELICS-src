Detailed Description

template<typename T>
class AirLock< T >

NOTES:: PT Went with unlocking after signaling on the basis of this page http://www.domaigne.com/blog/computing/condvars-signal-with-mutex-locked-or-not/ will check performance at a later time class implementing a airlock

this class is used to transfer an object from a thread safe context to a single thread so it can be accessed without locks
