Detailed Description

class implementing a Echo object, which will generate endpoint interfaces and send a data message back to the source at the with a specified delay

the Echo class is not threadsafe in general, don't try to use it from multiple threads without external protection, that will result in undefined behavior. setEchoDelay is threadsafe
