Detailed Description

template<class X>
class SimpleQueue< X >

class for very simple thread safe queue

uses two vectors for the operations, once the pull vector is empty it swaps the vectors and reverses it so it can pop from the back as well as an atomic flag indicating the queue is empty

Xthe base class of the queue
